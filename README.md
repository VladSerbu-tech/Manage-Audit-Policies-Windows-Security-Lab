# Windows Security Labs

This repository contains hands-on labs focused on **Windows Security, System Administration, and Cybersecurity fundamentals**.

The goal of this repository is to document practical exercises performed in virtual lab environments while learning how to configure and secure Windows systems.

## Labs Included

| Lab | Description |
|----|----|
| Manage Audit Policies | View and manage Windows audit policies, monitor security events, and understand how audit logs are stored |

## Skills Practiced

- Windows Security Configuration
- Audit Policy Management
- Command Line Administration
- Security Monitoring
- Event Log Analysis

## Tools Used

- Windows Server / Windows Client
- Command Prompt
- Group Policy
- Event Viewer
